# Simple-Windows-Keylogger
📡 Function: Sends victim's keyloggs to the remote attacker via a burner email. 

The objective of this project was to understand how keylogging-based malware functions from an attacker’s perspective, and more importantly, why such techniques can sometimes evade traditional endpoint defenses if not properly hardened.

Through this project, an ethical hacker can gain practical insight into: ⚙️ How keystroke capture mechanisms work at a low level ⚙️ Common persistence techniques abused by malware on Windows systems ⚙️ The limitations of signature-based detection methods

This experience reinforced a core principle: to defend effectively, you must first understand how attacks are built, deployed, and maintained.
